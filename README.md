# Song Illuminator

Song Illuminator connects current music playback to selected smart lights. Product and architecture decisions live in [`docs`](docs); the product application lives in [`app`](app).

The application is currently at the Milestone 1 foundation: a Go HTTP and GraphQL service, React frontend, Firestore wiring, and a Cloud Run-ready container and beta infrastructure definition.

See [`app/README.md`](app/README.md) for local development and verification commands.
